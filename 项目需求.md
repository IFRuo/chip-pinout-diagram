# 芯片引脚功能图绘制工具 - 项目需求

## 1. 项目概述

### 1.1 项目名称
芯片引脚功能图绘制工具

### 1.2 项目简介
一个用于绘制芯片引脚功能图的桌面应用程序，用户可在操作区输入引脚信息，系统自动生成带样式的SVG格式引脚功能图。

### 1.3 核心功能
- 根据用户输入绘制芯片引脚功能图（SVG格式）
- 支持读取/保存芯片引脚文件
- 支持功能组样式管理
- 支持多种封装形式（PID、QFN、QFP等）

---

## 2. UI/UX需求

### 2.1 主窗口布局

#### 2.1.1 基本信息栏（第一行）
- **芯片名称文本框**：输入芯片型号
- **封装类型选择框**：下拉选择封装形式
  - 两排引脚封装：PID、SOJ、SO、SOP、TSSOP
  - 四排引脚封装：QFP、LQFP、QFN、LCC
- **引脚数设置**：
  - 两排封装：总引脚数输入框
  - 四排封装：上、左、下、右四个引脚数输入框
- **样式管理按钮**：右对齐，点击弹出功能组样式管理窗口

#### 2.1.2 显示绘制区（左侧）
- 占窗口大部分区域
- 支持鼠标滚轮缩放
- 区域左上角显示全部功能组样式列表
- 中心为芯片引脚绘制区域

#### 2.1.3 操作区（右侧）
- **引脚功能分配区**：
  - 列表形式显示所有引脚
  - 每行包含：复选框1（显示/隐藏）、复选框2（占PIN/不占PIN）、引脚号、功能字段...
  - PIN00行特殊处理（中心焊盘）
  - 每行末尾有"+"按钮添加功能字段
  - 底部有"添加引脚"按钮
- **文件操作区**（底部）：
  - 读取按钮
  - 保存按钮
  - 输出图片按钮

### 2.2 弹出窗口

#### 2.2.1 功能组样式选择窗口
- 显示预设功能组样式列表
- 双击选择功能组并关闭
- 单击选中后可编辑或新建
- 由"样式管理"按钮打开时仅查看

#### 2.2.2 样式编辑窗口
- 编辑/新建功能组样式
- 包含：组名、字体、颜色、边框等参数
- 确认/删除按钮

### 2.3 交互规则

#### 2.3.1 引脚功能字段编辑
- 单击：进入文本编辑状态，回车保存并退出
- 双击：弹出功能组样式选择窗口
- 编辑为空：删除该功能字段

#### 2.3.2 功能组自动匹配
- 引脚功能添加后，自动匹配功能组
- 匹配依据：关联引脚功能名

## 3. 功能需求

### 3.1 核心功能

#### 3.1.1 芯片引脚图绘制
- 根据封装形式绘制芯片外形
- 绘制引脚焊盘
- 显示引脚号
- 显示引脚功能标签（带样式）

#### 3.1.2 封装形式支持
| 封装类型 | 引脚布局 | 焊盘位置 | 引脚数设置 |
|---------|---------|---------|-----------|
| QFP/LQFP | 四排 | 封装外形框外 | 上、左、下、右 |
| QFN/LCC | 四排 | 封装外形框内 | 上、左、下、右 |
| PID/SOJ/SO/SOP/TSSOP | 两排 | 封装外形框内 | 总引脚数 |

#### 3.1.3 引脚编号规则
- 左上角为第一脚
- 逆时针排列

### 3.2 文件操作

#### 3.2.1 读取文件
- 支持芯片引脚文件格式
- 支持简易芯片引脚文件格式
- 自动解析并填充UI

#### 3.2.2 保存文件
- 保存为芯片引脚文件
- 文件名格式：芯片名-封装类型-引脚数.txt

#### 3.2.3 导出图片
- 导出SVG格式
- 文件名格式：芯片名-封装类型-引脚数.SVG

### 3.3 功能组样式管理

#### 3.3.1 功能组样式参数
- 组名
- 字体
- 加粗/斜体
- 字号
- 字体颜色
- 圆角半径
- 边框线宽
- 边框颜色
- 背景颜色
- 关联引脚功能名列表

#### 3.3.2 功能组样式文件(.gn)
- 位置：preset/Groupingstyle.gn
- 启动时加载
- 修改后保存

### 3.4 中心焊盘(PIN00)规则
- 任何封装都可能出现中心焊盘
- 由PIN00行复选框决定是否显示
- 只以焊盘图+文本形式存在
- 无中心焊盘时显示芯片名称和封装类型

### 3.5 引脚可见性控制

#### 3.5.1 复选框1（显示/隐藏）
- 取消选择：隐藏该引脚的焊盘、PIN号、功能字段
- 图中仍占用一个引脚位置

#### 3.5.2 复选框2（占PIN/不占PIN）
- 取消选择：该引脚不再占用PIN序号
- 图中仍占用一个引脚位置
- 适用于空脚芯片（如单运放）

## 4. 数据格式

### 4.1 芯片引脚文件格式
```
/*芯片型号-封装类型-PIN数*/
组名1:参数
组名2:参数
01:[功能1,组名],[功能2,组名]
02:[功能1,组名],[功能2,组名],[功能3,组名]
```

### 4.2 简易芯片引脚文件格式
```
/*芯片型号*/
1:功能1/功能2/功能3
2:功能1/功能2
```

### 4.3 功能组样式文件格式(.gn)
```
组名:字体,加粗,斜体,字号,字体颜色,圆角半径,边框线宽,边框颜色,背景颜色,{关联引脚功能名1,关联引脚功能名2}
```
